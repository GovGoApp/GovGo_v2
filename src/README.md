# Frontend Real do GovGo v2

O frontend real da v2 passa a ser organizado a partir desta raiz.

O desenho completo da estrutura esta em `docs/ESTRUTURA_FRONTEND_V2.md`.

Regra de uso:

- `design/` continua sendo a referencia canonica;
- `src/` e onde a aplicacao real sera implementada;
- paginas entram em `src/pages/`;
- funcionalidades reutilizaveis entram em `src/features/`;
- tokens, theme, icons e primitives entram em `src/design-system/`.
